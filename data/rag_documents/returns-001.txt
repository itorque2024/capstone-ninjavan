Topic: Return Shipments

To return a parcel, contact the seller first to initiate the return. If the seller uses NinjaVan's return service, they will arrange a return pickup or drop-off and provide a return airway bill. Ensure the item is securely re-packaged before the driver arrives or before you drop it off. If the seller does not use NinjaVan's return service, you can arrange the return yourself using Ninja Flexi at https://delivery.ninjavan.co/c2c. For drop-off, visit any participating Ninja Point (800+ locations) with your return airway bill and government-issued ID.
