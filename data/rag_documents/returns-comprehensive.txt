Topic: Returns — Full Policy and Process

RETURN ELIGIBILITY:
- Return eligibility (timeframe, condition requirements) is set by the merchant/seller, not NinjaVan.
- NinjaVan handles the return logistics; the merchant decides whether to accept the return.
- Contact the seller first to confirm return eligibility before arranging shipping.

HOW TO INITIATE A RETURN:
1. Contact the seller to obtain return authorisation.
2. If the seller uses NinjaVan's return service: the seller will book the return pickup or provide drop-off instructions.
3. If the seller does NOT use NinjaVan: use Ninja Flexi at https://delivery.ninjavan.co/c2c to ship the return yourself.
4. Pack the item securely in original packaging if available.
5. Attach the return airway bill provided by the merchant.
6. Schedule a pickup or drop off at a Ninja Point.

RETURN PICKUP (merchant-initiated):
- NinjaVan rider collects the return parcel from your address.
- Pickup windows: between 9 AM and 10 PM on the selected date.
- The merchant or seller will advise if pickup is available for your return.

RETURN DROP-OFF:
- Drop off at any participating Ninja Point (800+ locations).
- Bring the return airway bill and government-issued ID.
- A receipt will be issued as proof of drop-off.
- Returns NOT accepted at 7-Eleven Ninja Points.
- Standard return size limit: under 5 kg and 80 cm combined (L+W+H) — check map for larger-item locations.
- Ninja Point locator: ninjavan.co/en-sg/support/find-ninja-point

NON-RETURNABLE ITEMS (common across most merchants):
- Perishables and fresh food
- Personal hygiene products
- Customised or personalised items
- Digital downloads and software once opened
- Items marked Final Sale by the merchant

RETURN TRACKING:
- Once your return is picked up or dropped off, you will receive a return tracking number.
- Track at ninjavan.co/en-sg/tracking using the return tracking number.

REFUND TIMELINE:
- Refunds are processed by the merchant, not NinjaVan.
- Once the merchant confirms receipt, typical timelines:
  - Credit/debit card: 5–10 business days
  - Bank transfer: 3–7 business days
  - Store credit: 1–3 business days

WRONG ITEM OR MISSING ITEM:
- Contact the merchant first.
- If no response within 3 business days, escalate by contacting NinjaVan support via shippercare_sg@ninjavan.co.
- NinjaVan may arrange re-delivery or facilitate a return at no additional cost.
